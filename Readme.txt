## 개요
링크 저장 모음 사이트

## 기능
1. 메인 화면 보여주기
2. 메인 화면 우측의 웹툰 바 보여주기 펼치고 접는 거
3. nav바를 통해 회원가입 페이지로 이동
4. 회원 가입 해보기(아이디 중복 상황 확인, 이메일 중복 검사)
5. 회원 가입 완료 후 로그인 페이지로 가서 로그인
6. 로그인 시 아이디 & 비번 틀려보기
7. 로그인 완료 후 nav바를 통해 게시글(board 페이지)로 이동
8. 게시글 눌러서 링크 이동 확인 및 조회수 증가 확인(새로 고침)
9. 게시글 수정 버튼을 통해 게시물 내용 수정하기
10. 게시글 카테고리 별 이동 (수정하기로 카테고리 수정된 곳으로 변경되었는지도 확인)
11. 게시글 url 수정 시 바뀐 url로 이동되는지 확인
12. 게시글 삭제 버튼을 눌러 게시글 삭제하기
13. 웹툰 상세 페이지로 이동하기
14. 로그아웃 버튼 누르기
